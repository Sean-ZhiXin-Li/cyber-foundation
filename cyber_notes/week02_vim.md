## Commands Learned

### Basic Editing

* `i` : enter insert mode
* `Esc` : return to normal mode
* `:w` : save file
* `:q` : quit Vim
* `:wq` : save and quit
* `:q!` : quit without saving

### Navigation

* `gg` : jump to the beginning of the file
* `G` : jump to the end of the file
* `5G` : jump to line 5

### Search

* `/keyword` : search for a keyword
* `n` : next match
* `N` : previous match

### Editing Operations

* `dd` : delete current line
* `yy` : copy current line
* `p` : paste below current line
* `u` : undo previous change
* `Ctrl+r` : redo change

---

## Biggest Lesson

Vim is not a normal text editor.

Instead of directly typing text, Vim uses different modes for editing and command execution.

Understanding the difference between Insert Mode and Normal Mode is the key to using Vim effectively.
